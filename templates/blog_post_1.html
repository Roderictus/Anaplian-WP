Desarrollo de un Dashboard Automatizado para Monitoreo de Parques Nacionales

Publicado el 22 de enero de 2025 por Rodrigo Franco

Anaplian comienza el año 2025 con avances en uno de nuestros proyectos públicos favoritos: el desarrollo de un dashboard automatizado para el monitoreo de Parques Nacionales.

Concebido como una herramienta multipropósito, nos enfocamos 1en el monitoreo de Áreas Naturales Protegidas (ANP) en México. Debido a que existen diferentes designaciones de ANP con variados grados de protección, decidimos concentrarnos en la categoría más restrictiva: Parques Nacionales.

Para obtener los shapefiles que definen los límites de estos parques, utilizamos la base de datos de Protected Planet. Los datos satelitales, con una resolución de 10 metros por píxel, fueron generados creando imágenes compuestas en Google Earth Engine—una excelente fuente de datos públicos que es fácilmente accesible mediante Java o Python.

Desarrollamos diversas funciones para producir series de tiempo de imágenes utilizando bandas infrarrojas, de agua y de vegetación. En los videos, utilizamos color verdadero, pero para el análisis de vegetación empleamos bandas adicionales para calcular cambios vegetativos. En este caso, utilizamos NDVI, aunque continuamos explorando técnicas de vanguardia en este campo.

Ventajas Clave de este Enfoque

  1. Metodologías aplicables a cualquier región del mundo: Esto las hace altamente comparables.
  2. Bajo costo de adquisición de imágenes: Optimización de recursos.
  3. Independencia de información generada oficialmente: Mayor flexibilidad y acceso.

Resumimos estas ventajas en tres cualidades fundamentales para la protección ambiental: comparabilidad, escalabilidad e independencia. Nos emociona estar en un punto en el tiempo donde la calidad de nuestras herramientas—tanto para el análisis como para la adquisición de datos—nos permite abordar este y muchos otros desafíos análogos.
